The goal is to build strong problem-solving skills before moving into more advanced topics like Data Structures, Algorithms, and backend development.

## ▶️ Running the Code

Clone the repository:

```
git clone https://github.com/sarikatsaliki/python-practice.git
```

Navigate to the folder:

```
cd python-practice
```

Run any file:

```
python filename.py
```

Example:

```
python largest_element.py
```
