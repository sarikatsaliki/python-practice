# 🐍 Python Practice

This repository contains Python programs I’ve written to strengthen my fundamentals.  
It is organized by topic to build a strong foundation before moving into advanced concepts like OOP, DSA, and backend development.

---

## 🗂 Folder Structure

### 📁 basics/
Exercises focused on core Python concepts such as lists, dictionaries, loops, and string manipulation.

- budget_tracker.py – Calculate total expenses and filter transactions  
- string_slicing.py – Practice reversing strings and slicing patterns  
- password_validator.py – Username and password validation logic  

---

### 📁 functions/
Practice with function design and advanced argument handling.

- highest_even.py – Find the largest even number in a list  
- super_func.py – Explore *args and **kwargs  
- imp_func.py – Practice map(), filter(), and reduce()  
- lambda_expressions.py – Squaring numbers and sorting using lambda functions  
- list_comprehensions.py – Implementing list comprehensions for cleaner logic  

---

### 📁 oop/
Practice with core Object-Oriented Programming concepts.

- encapsulation.py – Private variables, getters, and state modification  
- inheritance.py – Parent and child classes, using super()  
- dunder_methods.py – Customizing object behavior with __str__ and __len__
