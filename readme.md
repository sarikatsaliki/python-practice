# Python Practice

This repository contains small Python programs I solved to strengthen my fundamentals.

Topics covered so far:
- String slicing
- Loops and conditions
- Dictionaries
- Basic input validation

## Scripts

1. **password_checker.py** – checks if a password has at least 8 characters and contains a digit.
2. **alphabet_slicing.py** – prints a string backwards and every second letter.
3. **budget_tracker.py** – keeps track of expenses, shows total spent and highest expense.

Goal: Build strong core Python skills before moving to advanced topics.
